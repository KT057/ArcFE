# Front End Boilerplate

A boilerplate setup for front-end development with automated directory generation and essential commands.

---

## 🚀 **Cursor**

### Directory Auto-Generation

1. **Select the context:**

   - Choose `create-page-structure.mdc` as the context file.

2. **Run the command:**

   - To generate a directory structure, enter:

     ```
     Create {pageName}
     ```

   **Examples:**

   - `Create news` – Generates a `news` folder.
   - `Create news/post` – Generates a `post` folder inside `news`.

---

## 🛠️ **Commands**

### **Start Development Server**

Start the development server:

```bash
yarn start
```

---

### **Production Build**

Generate a production build:

```bash
yarn build
```

---

### **System Start**

Use this command to set up a temporary environment:

```bash
yarn sys:dev
```

---

### **Production Debug**

Debug the files generated by the `yarn build` command:

```bash
yarn debug
```

---

### **How to Lint and Format Fix**

Automatically fix lint and formatting issues:

```bash
yarn fix
```
